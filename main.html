360° + θ

Angles repeat every 360°. On the unit circle, 360°+θ lands on the same terminal point as θ.

current: 45°

360° + θ identities

  • sin(360° + θ) = sin θ
  • cos(360° + θ) = cos θ
  • tan(360° + θ) = tan θ
  • Also applies to sec, csc, cot (periodic)

Definitions

sin θ = opposite / hypotenuse • cos θ = adjacent / hypotenuse • tan θ = opposite / adjacent

Example

If θ = 45°, 360° + θ = 405°. The terminal point repeats; trig values are the same.
