About Space Hub

A modern, interactive learning platform built for presentations and education

Our Mission

Space Hub was created to provide an engaging, interactive platform for educational content. Combining beautiful space-themed design with practical learning tools, we aim to make education more accessible and enjoyable for students.

Key Features

  • Interactive unit circle visualization
  • Comprehensive mathematics resources
  • Project gallery with grid/list views
  • Fun educational games
  • Embedded presentations
  • Responsive, mobile-friendly design
  • Dark/Light theme support

Built With

Space Hub is built using modern web technologies including HTML5, CSS3, and JavaScript. We utilize GSAP for smooth animations, Canvas API for interactive graphics, and custom SVG icons for a crisp, scalable design.

Developer

Created by Sincere Bhattarai, a passionate student and developer. This project showcases skills in web development, design, and interactive programming, built specifically for school presentations and educational purposes.
